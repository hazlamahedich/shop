💡 How to Test the Demo

  • Start your development server: cd frontend && npm run dev
  • Click "Launch Interactive Demo" or navigate to http://localhost:5173/widget-demo
  • Use the control panel to switch between different features
  • Toggle between light and dark themes to see glassmorphism effect
  • Try voice input (requires microphone permission)
  • Click quick reply buttons to see responses
  • Drag the chat window to test smart positioning
  • Proactive Engagement: Move mouse to top of page and exit to trigger modal
  • Watch for proactive engagement triggers after 30 seconds (time on page)
  • Hover over product cards for animation effects
  • Click the chat bubble to show/hide the widget
