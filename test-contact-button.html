🧪 How to Test

  1. Open the widget - Click the chat button in bottom right corner
  2. Send a message - Type "hello" or any message
  3. Trigger handoff - Ask something that requires human assistance like "I need to speak to a human" or "transfer me to an agent"
  4. Check for Contact Us buttons - They should appear in the bot's response
  5. Test the buttons:
    • 📞 Phone: On mobile, will call; on desktop, will copy to clipboard
    • 📧 Email: Opens email client with subject line
    • 🔗 WhatsApp: Opens WhatsApp in new tab

🔍 Debugging Tips

  • Open browser DevTools (F12) → Console to see any errors
  • Check Network tab for API requests to /api/v1/widget/message
  • Look for contact_options in the API response
  • If buttons don't appear, the bot may not have triggered a handoff
